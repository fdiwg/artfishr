# artfish_new

artfish_new

## Usage

``` r
artfish_new(
  active_vessels = NULL,
  effort = NULL,
  effort_source = c("fisher_interview", "boat_counting"),
  active_days = NULL,
  landings = NULL,
  minor_strata = NULL
)
```

## Arguments

- active_vessels:

  active vessels

- effort:

  effort

- effort_source:

  effort source whether it's derived from -B1- (fishers interviews) or
  -B2- (boat counting)

- active_days:

  active days

- landings:

  landings

- minor_strata:

  minor_strata (to investigate further later)

## Value

the result of Artfish

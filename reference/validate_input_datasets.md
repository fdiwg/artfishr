# Validate input datasets

Validate input datasets

## Usage

``` r
validate_input_datasets(
  active_vessels,
  effort,
  effort_source = c("fisher_interview", "boat_counting"),
  active_days = NULL,
  landings
)
```

## Arguments

- active_vessels:

  active vessels

- effort:

  effort

- effort_source:

  effort source whether it's derived from -B1- (fishers interviews) or
  -B2- (boat counting)

- active_days:

  active days

- landings:

  landings

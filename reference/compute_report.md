# compute_report

compute_report

## Usage

``` r
compute_report(
  effort,
  effort_source,
  active_vessels,
  active_vessels_strategy,
  active_days,
  landings,
  minor_strata = NULL,
  validate = FALSE,
  progress_fn = NULL
)
```

## Arguments

- effort:

  effort

- effort_source:

  effort_source

- active_vessels:

  active_vessels

- active_vessels_strategy:

  active_vessels_strategy

- active_days:

  active_days

- landings:

  landings

- minor_strata:

  minor_strata. Default is `NULL`

- validate:

  validate

- progress_fn:

  a progress function with args (label, p). Default is `NULL`

## Value

the result of Artfish
